ヘルシート

  • 作品概要

    レシート画像で健康管理ができるWEBアプリケーションです。 現在iphoneアプリケーション版も制作中です。

  • 制作時期

    2019年11月~2020年1月

  • コンセプト

    一人暮らしの学生や社会人の方が、レシートで気軽に健康管理をできる

  • 工夫点

    OpenCVという画像処理ライブラリでレシート画像を加工した後に、OCR(光学文字認識)をすることでレシート画像を解析しています。

  • 制作環境と使用言語

    HTML5 CSS3 JavaScript Bootstrap Python Django SQL Swift Xcode
